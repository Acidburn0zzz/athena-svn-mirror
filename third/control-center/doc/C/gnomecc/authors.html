Authors

Gnome Control Center was written by Jonathan Blandford (<jrb@redhat.com>). Please send all comments, suggestions, and bug reports to the Gnome bug tracking database. (Instructions for submitting bug reports can be found on-line.) You can also use Bug Report Tool (bug-buddy), available in the Utilities submenu of Main Menu, for submitting bug reports.

This manual was written by Dave Mason (<dcm@redhat.com>) and Alexander Kirillov (<kirillov@math.sunsyb.edu>) with some help from other members of Gnome Documentation Project. Please send all comments and suggestions regarding this manual to the Gnome Documentation Project by sending an email to <docs@gnome.org>. You can also add your comments online by using the Gnome Documentation Status Table.
