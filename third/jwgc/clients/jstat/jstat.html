NAME

jstat - display Jwgc status

SYNOPSIS

jstat [ -h ] [ -d <flags> ]

DESCRIPTION

Jstat is used to display the current status of jwgc's connection to the Jabber server. It lists out the services available from the Jabber server you are connected to, some information about your connection to the Jabber server, and some various statistics.

Jstat understands the following options:

-h Displays usage information.

-d <flags> Debugging output. Run without <flags> for usage.
