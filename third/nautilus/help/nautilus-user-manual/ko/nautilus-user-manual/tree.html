나무 구조

나무 구조 보기를 사용하여서 컴퓨터의 파일과 폴더를 대략적으로 볼수 있습니다. 많은 사람들은 폴더를 선택하여 여는것보다 나무 구조 보기를 사용하여 탐색하여서 더 빠르게 찾습니다.

나무 구조 보기를 볼려면, 가장자리막대의 아래에 있는 나무 구조 탭을 클릭하십시오. 나무 구조 보기를 보지 않으려면 탭을 다시 누르십시오.

트리 보기의 탭이 보이지 않는다면, 가장자리 막대에서 오른 마우스 마우스 버튼을 누르고 나무구조를 선택하십시오. 사용자 수준이 초급으로 설정되어 있으면 나무 구조 보기로 설정할 수 없습니다.

시작 지점 - 트리의 맨 위 - 는 슬래쉬(/)로 보여지는 루트 디렉토리입니다. 루트 디렉토리를 열거나 컴퓨터의 모든 파일이나 폴더를 닫으려면 확장 삼각형을 누르십시오. 컴퓨터의 항목이 계층적으로 배열 됩니다. 루트 디렉토리에는 컴퓨터의 위치와 더불어 네트웍 위치도 나열 됩니다. (주의: 루트 디렉토리는 "/"로 표시되며 그곳에는 또한 "root"라는 이름의 디렉토리도 있습니다.)

이렇게 하세요: 나무 구조에서 폴더를 열고 닫으려면:

  • 트리 보기에서 폴더를 열고 닫으려면 삼각형을 클릭하십시오.

  • 오른쪽 패널에 폴더의 내용을 표시하려면, 트리에서 폴더의 이름을 클릭하십시오.
