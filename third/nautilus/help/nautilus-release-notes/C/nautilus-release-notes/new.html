What's New in Nautilus 1.0

  • The Nautilus window makes it easy for you to view and manage files on your computer, and visit sites on the Web.

  • The sidebar includes views of your directory hierarchy (tree), access to help and information sources, a list of recently-visited page, folder, and website locations, and space to record notes about specific folders and other items.

  • Nautilus offers various views of your file and folders, such as an icon view, a list view, and a music view for MP3 files.

  • For people who have a lot of MP3 files, extra features for music files and folders let you organize your MP3s into albums, attach custom icons, and more.

  • Nautilus includes a fast find feature that indexes your hard disk so you can quickly search for a file by content as well as file name, creation date, and other file attributes.

  • Nautilus has several customization features: You can change backgrounds, personalize icons, and switch themes.

  • Quick access to Eazel Services is available through the Nautilus toolbar.
