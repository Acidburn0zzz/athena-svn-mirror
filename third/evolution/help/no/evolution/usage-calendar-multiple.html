Multiple Calendars

Evolution permits you to have and maintain multiple calendars. This is useful if you maintain schedules for other people, if you are responsible for resource or room allocation, or if you have multiple personalities.

Example 1. Keeping Multiple Calendars

Keelyn, the office manager for a small company, has one calendar for her own schedule. On the local network, she maintains one for the conference room, so people know when they can schedule meetings. Next to that, she maintains a calendar that reflects when consultants are going to be on site, and another that keeps track of when the Cubs are playing.

To create a new calendar, select File->New->Folder. You'll need to tell the New Folder dialog that the new folder should be of the calendar type. You can place the calendar in any calendar folder and access it from the folder view. Alarms, configuration, and display for each calendar are separate from each other.
