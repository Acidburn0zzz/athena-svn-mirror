The Contact Editor Menus

This window has only one menu: the file menu.

Contact Editor File Menu

This menu contains five items:

Save

Save As

Allows you to save the contact as an external file in the VCard format.

Print

Prints the current contact.

Delete

Deletes the current contact.

Close

Closes the Contact Editor without saving.
