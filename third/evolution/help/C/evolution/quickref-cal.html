Calendar

Create a new Appointment:

File->New->Appointment or Ctrl+Shift+A

Create a new Task:

File->New->Task or Ctrl+Shift+T

New Appointments Fast: Click on any blank spot in the calendar and start typing to create a new appointment entry.
