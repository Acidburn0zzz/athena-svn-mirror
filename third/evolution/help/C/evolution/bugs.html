Appendix B. Known bugs and limitations

Ximian Evolution bug tracking is done at the Ximian bug tracking System. You may use that, or the GNOME Bug Report Tool (known as bug-buddy at the command line) if you find bugs or would like to request new features.

If you need additional help with Ximian Evolution, please visit the Ximian support site at ximian.com/support.

A number of often-requested features will not make it into Ximian Evolution 1.0, but we plan to include them for version 1.1. Some of them are:

Direct import of Microsoft Outlook contact cards
Synchronization of email with handheld devices
S/MIME support
Read foreign mail folders without importing them
Import of WINMAIL.DAT attachments
Playing a sound to indicate new mail arriving

A more complete list of feature requests and other issues with Evolution is available online in the Ximian bug tracking system.
