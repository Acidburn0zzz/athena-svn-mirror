GNOME Canvas Library Reference Manual

GnomeCanvasRect

Name

GnomeCanvasRect -- Rectangle canvas item

Synopsis

#include <libgnomecanvas/libgnomecanvas.h>


struct      GnomeCanvasRect;

Object Hierarchy

  GObject
   +----GtkObject
         +----GnomeCanvasItem
               +----GnomeCanvasShape
                     +----GnomeCanvasRE
                           +----GnomeCanvasRect

Description

A canvas item for drawing rectangles and squares. The parameters are defined in the parent classes including GnomeCanvasRE which is shared with GnomeCanvasEllipse items as well.

Details

struct GnomeCanvasRect

struct GnomeCanvasRect;

This structure should not be accessed directly. Use the object's properties for querying or changing its parameters.
