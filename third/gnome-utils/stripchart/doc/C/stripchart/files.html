Configuration and Parameter Files

The stripchart program determines the parameters to display by reading a configuration file. The stripchart program will look for a configuration file in the following order:

  • first, for a file specified on the command line,

  • then, for a file named stripchart.conf in the current working directory,

  • then, for a file named .stripchart.conf in the users home directory,

  • then, for a file named /etc/stripchart.conf,

  • and finally, for a file named stripchart.conf in the CONFDIR directory, which will usually be either /usr/share/stripchart/ or /usr/local/share/stripchart/.

If no configuration file is found, the program displays an error message and exits.

The canned parameters that are displayed in the Parameter Editor window are read from the stripchart.param file in the {prefix}/share/stripchart directory.
