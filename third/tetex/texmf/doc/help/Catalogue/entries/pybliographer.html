pybliographerA tool for managing bibliographic databases.
Pybliographer currently supports BibTeX files for reading and writing, and can also read Ovid files (from ovid.com). Support for Refer is available. This bibliographical manager can be used for searching, editing, reformatting, etc. It provides a simple framework that provides easy to use python classes and functions, and therefore can be extended to any usage (generating HTML pages according to bibliographic searches, etc). As an example, a simple tool called pybcheck is provided with the package to check for the validity of bibtex files. In addition to the scripting environment, a graphical GNOME interface is available, providing editing capabilities in addition to a nice hierarchical search mechanism. This package was implemented by Frederic Gobry and catalogued for CTAN.
