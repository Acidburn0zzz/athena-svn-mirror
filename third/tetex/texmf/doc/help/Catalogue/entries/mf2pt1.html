mf2pt1Produce PostScript Type 1 fonts from Metafont source
mf2pt1 facilitates producing PostScript Type 1 fonts from a Metafont source file. It is not, as the name may imply, an automatic converter of arbitrary Metafont fonts to Type 1 format. mf2pt1 imposes a number of restrictions on the Metafont input. If these restrictions are met, mf2pt1 will produce valid Type 1 output. (Actually, it produces "disassembled" Type 1; the t1asm program from the t1utils suite will convert this to a true Type 1 font.) This package was implemented by Scott Pakin and catalogued for CTAN.
