hyphenation-greekHyphenation patterns for ancient and modern Greek.
Contains the hyphenation patterns for ancient Greek and modern Greek in polytonic (multi-accent) and monotonic (uni-accent) systems. The hyphenation patterns will work perfectly with the "greek" option of "babel" or Dryllerakis' GreeKTeX. For any other Greek package, the patterns will have to be re-coded. A brief description of the files is given in the file "readme.txt". This package was implemented by Dimitrios Filippou and catalogued for CTAN.
