spanishbSupport for the Spanish language in Babel.
This spanish style for babel provides the functionality intented for the Spanish language when babel 3.7 comes to light. Very likely, its implementation will change in babel 3.7 because there are parts of code for it to work with 3.6; thus, it should be considered neither part of babel 3.7 nor babel 3.6.
