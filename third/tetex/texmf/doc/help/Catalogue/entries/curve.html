CurVeA LaTeX2e class for making curriculum vitae.
CurVe is a class package for writing a CV. It provides a set of commands to create rubrics, entries in these rubrics etc. CurVe will then format the CV (possibly splitting it onto multiple pages, repeating the titles etc), which is usually the most painful part of CV writing. Another nice feature of CurVe is its ability to manage different CV "flavors" simultaneously. It is in fact often the case that you want to maintain slightly divergent versions of your CV at the same time, in order to emphasize on different aspects of your background. CurVe also comes with support for AUC-TeX. This package was implemented by Didier Verna and catalogued for CTAN.
