xmltexSupport for parsing XML documents.
This package provides an implementation of a parser for documents matching the XML 1.0 and XML Namespace Recommendations. In addition to parsing commands are provided to attatch TeX typesetting instructions to th evariuos markup elemenets as they are encounted. Sample files for typesetting a subset of TEI, MathML, are included. Element and Attribute names, as well as character data, may use any characters allowed in XML, using utf-8 or a suitable 8bit encoding. This package was implemented by David Carlisle and catalogued for CTAN.
