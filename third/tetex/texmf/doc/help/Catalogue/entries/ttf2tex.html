ttf2texUse TrueType fonts with teTeX.
ttf2tex is a Bash script which will create all files neccessary to use TrueType fonts with teTeX from a set of *.ttf files. It will sort them, build the map files required by ttf2pk and pdftex, and optionally install them. It is designed for UNIX/Linux systems running teTeX. This package was implemented by Philipp Lehman and catalogued for CTAN.
