prelim2eAllows the marking of preliminary versions of a document.
Puts some text below the normal page content. The default text marks the document as draft and puts a timestamp on it. Can be used together with e.g. the vrsion, rcs and rcsinfo packages. Uses the everyshi package and can use the scrtime package from the koma-script bundle. This package was implemented by Martin Schroder and catalogued for CTAN. See also everyshi count1to koma-script
