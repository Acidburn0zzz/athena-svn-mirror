matlabwebLiterate programming system for Matlab.
A literate programming system for the Matlab language. Similar to CWEB, created with a slightly modified version of the Spider system. Can be used with plain TeX or LaTeX, the latter with help from the webfiles package. This package was implemented by Mark Potse and catalogued for CTAN.
