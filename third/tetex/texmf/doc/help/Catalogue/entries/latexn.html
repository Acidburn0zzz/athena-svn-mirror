latexnRun LaTeX as many times as needed.
A csh script to run LaTeX as many times as needed (and hopefully no more) on a given file to resolve cross references, and to ensure that the table of contents and index (if any) are up-to-date. This package was implemented by John Collins and catalogued for CTAN. See also latexmk
