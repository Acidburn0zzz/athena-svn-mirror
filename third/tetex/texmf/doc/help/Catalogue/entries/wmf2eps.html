wmf2epsWindows meta file conversion.
A program to simplify MS-Windows Metafile Graphics (WMF) containing either vector-images or bitmaps into Encapsulated PostScript format having tight bounding-boxes (WinNT and Win95) at exactly the same size as the WMF-originals. This package was implemented by Wolfgang Schulter and catalogued for CTAN.
