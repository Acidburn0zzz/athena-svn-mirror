aeVirtual fonts for PDF-files with T1 encoded CMR-fonts.
A set of virtual fonts which emulates T1 coded fonts using the standard CM fonts. The package is called AE fonts (for Almost European). The main use of the package is to produce PDF files using versions of the CM fonts instead of the bitmapped EC fonts. This package was implemented by Lars Engebretsen and catalogued for CTAN.
