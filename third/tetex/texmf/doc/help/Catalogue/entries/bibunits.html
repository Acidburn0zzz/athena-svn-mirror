bibunitsMultiple bibliographies in one document.
A style to generate separate bibliographies for different units (parts) of the text (chapters, sections or bibunit-environments). The style will separate the citations of each unit of text into a separate file to be processed by BibTeX. The global bibliography section produced by LaTeX can also appear in the document and citations can be placed in both at the same time. The package is compatible with koma-script and frenchb. This package was implemented by Jose Alberto and Thorston Hansen and catalogued for CTAN. See also bibtopic chapterbib
