pkfixReplace pk fonts in PostScript with Type 1 fonts.
The perl script pkfix looks for DVIPSBitmapFont comments in PostScript files, generated by not too old dvips, and replaces them by the type 1 versions if possible. This package was implemented by Heiko Oberdiek and catalogued for CTAN.
