bibwebAutomatically retrieve bibliography from MathSciNet.
A utility to automatically retrieve mathematical bibliographical information, in BibTeX format, from the American Mathematical Society's MathSciNet database. More precisely, it acts as a front-end to BibTeX; it converts any citations that BibTeX can't find into queries to the MathSciNet database, it carries out those queries, and returns the answers in BibTeX format. Included in the package are: the perl script bibweb, documentation (in various formats), and a test file. This package was implemented by John H. Palmieri and catalogued for CTAN.
