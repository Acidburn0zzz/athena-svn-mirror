makeglosInclude a glossary into a document.
A LaTeX package to include a glossary into a document. The glossary must be prepared by an external program, like xindy or makeindex, in the same way that an index is made. This package was implemented by Thomas Henlich and catalogued for CTAN. See also nomencl glosstex gloss
