sf298Standard form 298.
A LaTeX package for generating a completed standard form 298 (Rev. 8-98) as prescribed by ANSI Std. Z39.18 for report documentation as part of a document delivered, for instance, on a U.S. government contract. This package was implemented by Steven Douglas Cochran and catalogued for CTAN.
