ps2epsProduce Encapsulated PostScript from single page PostScript.
Produce Encapsulated PostScript Files (EPS/EPSF) from usual one-paged Postscript documents. It calculates correct Bounding Boxes for those EPS files and filters some special PostScript command sequences that can produce errorneous results on printers. EPS files are needed for including (scalable) graphics into TeX documents. Other programs like "ps2epsi" don't always calculate the correct bounding box (because the values are put on the PostScript stack which may get corrupted by bad PostScript code) or rounded it off so that clipping the EPS cut some part of the image. Therefore ps2eps uses a resolution of 144 dpi to get the correct bounding box. ps2eps needs perl, ghostscript and an ANSI-C compiler if your platform is not Linux, Solaris, Digital Unix or Windows 2000/9x/NT (binaries included).
