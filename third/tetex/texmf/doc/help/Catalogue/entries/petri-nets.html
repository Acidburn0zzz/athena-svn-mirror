Petri-nets packagesTeX/LaTeX packages for drawing Petri nets
Petri-nets is a set of TeX/LaTeX packages about Petri nets and related models. One package allows to draw Petri-nets in PostScript documents. One other defines macros related to PBC, M-nets and B(PN) models. A last package just gathers together the two previous. This package was implemented by Franck Pommereau and catalogued for CTAN.
