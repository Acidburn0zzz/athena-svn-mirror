cwebbinCWEB for ANSI-C/C++ compilers on UNIX/Linux, MS/Windows, and Amiga.
A highly portable and slightly extended version of cweb for Unix, MS-Windows32, and Amiga (and possibly other operating systems). TeX macros and CWEB macros are provided for German, French, and Italian program documentation on any machine. Comes with binaries. Note that the file cwebbin-p14.tar.gz contains Amiga binaries that are not found in later versions. This package was implemented by Andreas Scherer and catalogued for CTAN.
