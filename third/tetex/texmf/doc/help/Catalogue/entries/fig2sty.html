fig2styGenerate a layout in fig and then insert text into it in LaTeX.
fig2sty allows you to generate fancy layouts with LaTeX. The basic idea is to draw layout definitions interactively with XFig and transform this definition to a LaTeX style file. You can then use LaTeX to typeset your text into arbitrarily shaped polygons (frames) within the layout. This package was implemented by M. Rohner and catalogued for CTAN.
