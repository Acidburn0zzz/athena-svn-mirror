bakoma-malvernTeX for MS-Windows for electronic documents.
Malvern is a sanserif font intended mainly for non-technical documents. It does not blend particularly well with the Computer Modern fonts. Malvern is a font family designed and implemented in MF by P. Damian Cugley. Conversion to PostScript Type 1 font format has been done by Basil K. Malyshev. It is distributed as a MSWindows exe file which will install the fonts under already installed BaKoMa TeX 2.11 and later. This package was implemented by Basil Malyshev and catalogued for CTAN.
