xl2latexConvert Excel (97 and above) tables to LaTeX tabulars.
This Excel to LaTeX converter exports the current selection as a LaTeX file which can be included via \input in your actual document. It supports multicolumn cells (only horizontal), borders, font styles (bold, italic), colored cells and text, LaTeX math in cells and cells with wrapped text. This package was implemented by Ronny Buchmann and catalogued for CTAN.
