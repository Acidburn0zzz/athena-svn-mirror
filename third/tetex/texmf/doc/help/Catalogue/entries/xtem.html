xtemAn X11 TeX menu built on Tcl/Tk.
Provides a simple and comfortable graphical user interface to control file and directory selection, directory creation, vi, emacs, TeX, LaTeX, previewing, etc. Written in Tcl/Tk. This package was implemented by Guenther Lamprecht and Wolfhard Lotz and Roland Weibezahn and catalogued for CTAN.
