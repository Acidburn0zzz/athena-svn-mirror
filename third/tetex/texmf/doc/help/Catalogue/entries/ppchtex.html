ppchtexA package that can be used to typeset chemical formulas.
The package is a separate module of the context macro package for TeX (context is a full featured, parameter driven macro package, which fully supports advanced interactive documents). It is available as the file cont-ppc.zip. This package was implemented by Hans Hagen and catalogued for CTAN.
