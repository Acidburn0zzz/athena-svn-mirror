skakTypeset chess games.
This package can be used to typeset chess games using PGN and show diagrams of the current board in the document. The package builds on work by Piet Tutelaers - the main novelty is the use of PGN for input instead of the more cumbersome coordinate notation (g1f3 becomes the more readable Nf3 in PGN). This package was implemented by Torben Hoffmann and catalogued for CTAN.
