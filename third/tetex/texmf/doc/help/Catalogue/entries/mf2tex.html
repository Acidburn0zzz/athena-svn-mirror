mf2texNo caption.
A package of macros in TeX and MetaFont which allows a user to easily add labels to MetaFont sources (text or mathematics). Normally you can only draw in MetaFont and all labels you must write into the TeX source before (or after) inserting the picture. This package allows you to write TeX labels straight into the MetaFont source and to generate corresponding TeX source. This package was implemented by Robert Spalek and catalogued for CTAN.
