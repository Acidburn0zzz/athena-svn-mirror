mt11pNo caption.
A package to use the MathTime and MathTimePLUS (``MathTime complete'') fonts in LaTeX2e. Everything is included, incl. (patched) font metrics, except, of course, the fonts themselves. The package cooperates with the AMS packages (amsmath, amssymb, etc.) T1 and OT1 encodings (as operators fonts!) are fully supported. It does not assume you own any other commercial (non-resident) fonts. This package has no connection with the ``mathtime'' package by Frank Mittelbach and David Carlisle, commissioned by Y&Y. This package was implemented by Drahoslav Lim and catalogued for CTAN.
