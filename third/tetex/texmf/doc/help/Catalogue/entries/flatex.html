flatexNo caption.
A C program to flatten a LaTeX file into a single file, by explicitly including the files inclued by \include and \input commands. Also, if BibTeX is being used, then includes the bbl file into the resulting file. Thus, creating a stand alone LaTeX file that can be emailed to someone else. This package was implemented by Sariel Har-Peled and catalogued for CTAN. See also flatten
