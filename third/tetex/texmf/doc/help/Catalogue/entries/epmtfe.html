epmtfeTeX environment for OS/2.
The EPM TeX Front End is an additional module for the OS/2 `Enhanced Editor' (EPM), v6.03, turning the editor into a powerful integrated TeX environment that can be used in conjunction with emTeX or another OS/2 TeX system. This package was implemented by Walter Schmidt and catalogued for CTAN.
