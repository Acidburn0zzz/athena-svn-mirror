expdlistExpanded description environments.
The expanded description environment provides additional features to the LaTeX description environment. It supports changing the left margin. With \listpart there is a new command available which makes it possible to break a list for a comment without touching any counters. This package was implemented by Wolfgang Kaspar and catalogued for CTAN.
