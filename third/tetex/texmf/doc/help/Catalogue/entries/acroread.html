acroreadView pdf formatted files.
Acrobat (acroread) is a tool from Adobe for reading Adobe PDF files. It is available for a variety of architectures, and can be used as a plugin for Netscape. Adobe's distill (you have to buy that one) can be used to produce PDF files from PostScript. This provides a very useful way of accessing PostScript documents on the Web since PDF compresses the document internally. Best to use PostScript fonts in your LaTeX documents to make the most of it though. This package was implemented by Adobe and catalogued for CTAN. See also acrotex
