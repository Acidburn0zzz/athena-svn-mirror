subfloatSub-numbering for figures and tables.
This package enables sub-numbering of different floats (figures and tables) similar to the subequations-environment of the amsmath package. It is not the same as the subfigure package which generates sub-figures within one normal figure. This package was implemented by Harald Harders and catalogued for CTAN.
