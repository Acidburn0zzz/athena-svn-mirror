preview-latexPreview equations in Emacs.
Preview-latex allows appropriately selected parts of a LaTeX document to be formatted and displayed within your Emacs editor, allowing you to view what it looks like whilst still allowing you to edit it. This package was implemented by David Kastrup and catalogued for CTAN.
