psripExtracts images from PostScript files.
A Perl-script to extract images from PostScript files. The images are saved into the current directory if no other directory is given with the -d parameter. If there is no BoundingBox specified in the extracted image you have to insert it by hand. This package was implemented by Christian Lackas and catalogued for CTAN.
