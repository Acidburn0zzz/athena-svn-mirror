vrulerNo caption.
A package for adding a vertical numbering to the general text so that the text can be properly referenced. The vertical ruler can be scaled and moved freely. Supports LaTeX and plain TeX. This package was implemented by Zhuhan Jiang and catalogued for CTAN.
