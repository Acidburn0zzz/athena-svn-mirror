isodateTune the output format of the \today command.
This package provides four commands to switch the output format of the \today command: \isodate switches to yyyy--mm--dd; \numdate switches to dd.\,mm.~yyyy, dd/mm/yyyy, or mm/dd/yyyy; \shortdate switches to dd.\,mm.\,yy, dd/mm/yy, or mm/dd/yy; TeX format yyyy/mm/dd; and \origdate switches to the original LaTeX format (which is language dependent). The package contains two additional commands to print a date given as an argument using the actual date format for output: \printdateiso{yyyy-mm-dd} and \printdatenumger{dd.mm.yyyy}.
