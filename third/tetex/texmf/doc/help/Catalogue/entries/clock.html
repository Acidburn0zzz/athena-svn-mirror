clockGraphical and textual clocks for TeX and LaTeX.
Features graphical clocks (with a classical 12h dial and two hands) and text clocks (in 24h format) which can show system time or any time the user desires. Works with both TeX and LaTeX. The clock faces (appearances of the dial) are easily expandable. This package was implemented by Oliver Corff and catalogued for CTAN.
