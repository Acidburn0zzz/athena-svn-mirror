briefGerman letter style.
German private letter class with address boxes for window envelopes, a hook to put a figure or any kind of box in the top right corner, fold-marks (option falt), setting the text in informal \tt (option tt), configurable by a configuration file. This package was implemented by Stephan Boettcher and catalogued for CTAN. See also akletter dinbrief envlab formlett
