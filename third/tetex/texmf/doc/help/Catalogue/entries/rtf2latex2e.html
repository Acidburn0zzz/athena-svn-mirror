rtf2latex2eConvert Rich Text Format (RTF) files to LaTeX2e.
Converts Rich Text Format (RTF) files to LaTeX2e. It has support for figures and tables. Equations are read as figures. It can the handle the latest RTF versions from Microsoft Word 97/98/2000, StarOffice, and other word-processors. It has support for tables, figures, and to some extent equations. It runs on Macintosh, Linux, Unix, Windows. This package was implemented by Ujwal Sathyam and catalogued for CTAN.
