mmafmFont metrics for multiple-master font.
Creates an AFM file (font metrics) corresponding to an instance of a multiple-master font by interpolating at a given point in a multiple master's design space. It reads the AMFM and AFM files distributed with the font. This package was implemented by Eddie Kohler and catalogued for CTAN. See also mmpfb
