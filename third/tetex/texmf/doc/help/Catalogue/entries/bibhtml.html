bibhtmlBibTeX support for HTML files.
Bibhtml consists of a Perl script and a BibTeX style file, which together allow you to compile a bibliography for a collection of HTML files. The references in the text are linked directly to the corresponding bibliography entry, and if a URL is defined in the entry within the BibTeX database file, then the generated bibliography entry is linked to this. The BibTeX style file is plainhtml.bst, and is derived from the standard plain.bst. This package was implemented by Norman Gray and catalogued for CTAN.
