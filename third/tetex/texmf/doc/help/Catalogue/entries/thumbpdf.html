thumbpdfThumbnails for pdfTeX and dvips/ps2pdf.
Provides support, using Perl, for thumbnails in pdfTeX and dvips/ps2pdf, using ghostscript to generate the thumbnails which get represented in a TeX readable file that is read by the package thumbpdf.sty to automatically include the thumbnails. Works with both plain TeX and LaTeX. This package was implemented by Heiko Oberdiek and catalogued for CTAN.
