registerTypeset programmable elements in digital hardware (registers).
This package is designed for typesetting the programmable elements in digital hardware, i.e., registers. Such registers typically have many fields and can be quite wide; they are thus a challenge to typeset in a consistent manner. Register is similar in some aspects to the bitfield package by Reuben Thomas. Anyone doing hardware documentation using LaTeX should examine both packages. Register requires a fairly recent version of the float package. This package was implemented by Matthew Lovell and catalogued for CTAN.
