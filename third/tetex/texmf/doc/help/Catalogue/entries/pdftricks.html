pdftricksSupport for pstricks in pdftex.
The psTricks macros cannot be used with pdfTeX, since pstricks uses PostScript arithmetic, which isn't part of PDF. This package circumvents this limitation so that the extensive facilities offered by the powerful psTricks package can be made use of in a pdfTeX document. This is done using the shell escape function available in the web2c TeX compiler (limiting the use of this package to the web2c implementation of TeX). This package was implemented by C. V. Radhakrishnan and C. V. Rajagopal and catalogued for CTAN.
