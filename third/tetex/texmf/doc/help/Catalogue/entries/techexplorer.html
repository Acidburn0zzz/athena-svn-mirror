techexplorerNetscape Plugin for viewing TeX and LaTeX sources.
A plugin for Netscape which allows you to view TeX and LaTeX source files in Netscape as if you were using a dvi viewer, available for MS-Windows 95/NT, Sun Solaris, SGI IRIX, and IBM AIX. This package was implemented by an author unknown and catalogued for CTAN. See also ltoh latex2html hyperlatex pdftex
