emTeX/TDSTeX for OS/2.
An emtex-based TeX system for OS/2 with a TDS-compliant directory structure. The distribution contains a full texmf directory tree. It comes with an installation script that sets up a comprehensive, ready-to-run (La)TeX system, including full PostScript support. This package was implemented by Walter Schmidt and catalogued for CTAN.
