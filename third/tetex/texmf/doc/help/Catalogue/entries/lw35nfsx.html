lw35nfsxNo caption.
LaTeX psnfss support for the 35 printer resident PostScript fonts using ly1 text font encoding, employing the Berry names. This is similar to the existing lw35nfss.zip support using T1/TS1, but much simpler (lw35nfsx.zip is about 150k bytes while lw35nfss.zip is about 800 kbytes). It includes needed additions to dvips's psfonts.map, TFM metric files for TeX, FD font definition files, and STY files for LaTeX. This package was implemented by Berthold K. P. Horn and catalogued for CTAN.
