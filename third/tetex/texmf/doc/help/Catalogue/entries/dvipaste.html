dvipasteDVI manipulation.
A program designed to produce files that conform to the TeX specifications for dvi files, even though they may actually be too big to be produced with normal implementations of TeX (because of limitations on memory size). More particularly, it allows the material appearing on individual pages of a `secondary' file to be inserted into a `main' file. This package was implemented by an author unknown and catalogued for CTAN.
