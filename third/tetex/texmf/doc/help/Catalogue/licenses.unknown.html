The TeX Catalogue License Definitions: UNKNOWN

Unknown Status

The license status of the software is unknown and has to be assumed to be non-free. This is the default for any software for which the license status is not known within the Catalogue. The software may be free or nonfree but we've not been informed and as yet have not had the resources to check it out.

Please check the software's license before using the software.

Some software has no copyright statement nor license. By default, this is often regarded as `all rights reserved', hence such software is non-free and placed into this unknown category.

Further information about the licenses identified in the TeX Catalogue is available.
