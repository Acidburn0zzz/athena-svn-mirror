The TeX Catalogue License Definitions: NOSOURCE

No Source Available

This software is supplied in binary format and the source code for it is not made available. This renders the software non-free in the sense of the Debian Free Software Guidelines and many other free software advocates. The license may allow free use and distribution of the software, but modification is generally not possible. It may have other restrictions.

Please check the software's license before using the software.
