How do I run the GNOME Panel?

The GNOME panel is often started from gnome-session, if it is not, you can just type panel at any command prompt to start the panel.

Add things to the panel by either the Foot's Panel menu, or by the menu you get by right clicking on the panel. The GNOME User's Guide has an entire Chapter on how to use the Panel.
